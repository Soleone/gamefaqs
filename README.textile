h1. GameFAQs library

Access information about all games (any platform) from GameFAQs.
You can search for games by title and platform, and then view _Reviews_, _FAQs_, _Cheats_ for it.

h2. Usage example
<pre>
<code>
  require 'gamefaqs'
  
  # Search for a game containing the two words on the Nintendo DS
  castlevania = GameFaqs::Search.game("Castlevania Ecclesia", "DS")
  # get all reviews for this game
  reviews = GameFaqs::Review.all_for_game(castlevania)
  # get the first review in the list
  review = reviews.first
  
  # Score in the format 9/10
  review.score
  review.title
  review.author
</code>
</pre>
